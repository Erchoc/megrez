# 二值化（Binarization）

**二值化**是**图像分割**的一种最简单的方法。二值化可以把**灰度图像**转换成**二值图像**。把大于某个临界灰度值的像素灰度设为**灰度极大值**，把小于这个值的像素灰度设为**灰度极小值**，从而实现二值化。

根据**阈值**选取的不同，二值化的算法分为**固定阈值**和**自适应阈值**。 比较常用的二值化方法则有：**双峰法**、**P参数法**、**迭代法**和**OTSU法**等。
