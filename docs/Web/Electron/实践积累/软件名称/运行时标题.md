# 运行时标题

运行时标题来自 `html` 中的**网页标题**，按照具体技术栈配置即可。

## vue项目配置

> ![tip|label: 提示]
> 采用 `vue3.0` + `webpack` 构建工程。

修改 `vue.config.js` 配置文件，如下所示：

```js
module.exports = {
    chainWebpack: config => {
        config.plugin('html').tap(args => {
            args[0].title = 'MQTT模拟测试工具';
            return args;
        });
    }
}
```
