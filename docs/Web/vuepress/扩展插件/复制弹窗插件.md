# 复制弹窗插件

## 文档

* [npm](https://www.npmjs.com/package/vuepress-plugin-nuggets-style-copy)

## 安装

```shell
$ yarn add -D vuepress-plugin-nuggets-style-copy
```

## 配置

在 `.vuepress/config.js` 文件中配置 `plugin` 字段，如下所示：

```javascript
plugins: [
    [
        'vuepress-plugin-nuggets-style-copy',
        {
            copyText: '复制代码',
            tip: {
                content: '复制成功'
            }
        }
    ]
],
```
