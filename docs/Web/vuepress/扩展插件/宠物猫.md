# 集成宠物猫
* [GitHub地址](https://github.com/QiShaoXuan/vuepress-plugin-cat)

## 安装

```shell
$ yarn add -D vuepress-plugin-cat
```

## 配置

在 `.vuepress/config.js` 文件中配置 `plugin` 字段，如下所示：

```javascript
plugins: [
  'vuepress-plugin-cat'
],
```

::: warning 注意
这个插件是基 `于vuepress 1.x` 的，现在已经是 `2.x` 版本了，要注意后面是否可用。
:::
