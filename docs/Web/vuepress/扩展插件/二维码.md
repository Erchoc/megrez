# 二维码
* [Github地址](https://github.com/openHacking/vuepress-plugin-qrcode)
  

## 安装

```shell
$ yarn add -D vuepress-plugin-qrcode
```

## 配置

修改 `docs/.vuepress/config.js` 文件，增加如下内容

```javascript
module.exports = {

    plugins: [
      [
        'qrcode',
        {
          labelText: '二维码'
        }
      ]
    ]

}
