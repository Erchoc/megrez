# 图片放大配置

## 文档地址

* [社区文档地址](https://vuepress-community.netlify.app/zh/plugins/zooming/#%E5%AE%89%E8%A3%85)

## 安装

```bash
$ yarn add -D vuepress-plugin-zooming
```

## 配置

修改 `docs/.vuepress/config.js` 文件，增加如下内容

```javascript
module.exports = {
    plugins: [
        'vuepress-plugin-zooming'
    ]
}
```
