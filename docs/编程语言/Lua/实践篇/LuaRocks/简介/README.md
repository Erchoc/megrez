# 简介

`LuaRocks` 是 `Lua` 的包管理器，基本上大部分现代语言都有包管理器， `Python` 有 `pip` 、 `Java` 有 `Maven` 、 `NodeJS` 有 `npm` 等等。只不过 `Lua` 很多时候被用于**嵌入式**开发，导致资源紧张，无法使用规模庞大的第三方包。因此， `LuaRocks` 名声不显。

`Lua` 语言出来的很早了，最开始的定位也是一个轻量级的语言，所以并没有像 `Python` 和 `NodeJS` 那样自带原生的包管理器， `LuaRocks` 也是后来才逐渐发展起来的，并且逐步称为主流。

> [!tip|label: 提示]
> 尽管如此， `LuaRocks` 在一些嵌入式领域仍然名声不显，例如低成本的嵌入式模块开发、可以跑 `Lua` 虚拟机的 `MCU` 开发等等。只有在游戏开发， `OpenResty` 等资源丰富的领域才被广泛接纳使用。

在语言的发展演化中， `LuaRocks` 也不是 `Lua` 生态中唯一的包管理器，比如已经被淘汰的 `LuaDist` 。因此，在使用这个工具的时候要注意其生态，可能这个生态也会处于一种停滞状态，如果遇到这类情况，慎用！
